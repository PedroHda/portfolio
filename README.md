# Portfolio Pedro Domiciano

Portfolio em Next.js com TypeScript, estruturado a partir do currículo.

## Rodar localmente

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Download do leitor de Markdown

O card do projeto já está pronto. Para ativar o botão de download, coloque o
executável neste caminho:

```text
public/downloads/leitor-markdown.exe
```

Quando o arquivo existir antes da execução/build, a página exibe o botão
`Baixar .exe`.
